# asapp-cc
## How to Run
1. Clone repo, go to cloned directory
2. Install dependencies ($yarn)/($npm install)
3. Run the application ($yarn start)/($npm start)

## Tech Stack
- Javascript, HTML, CSS, React
- Babel, Webpack
- Node.js/Express, GraphQL, Apollo
- ESLint
